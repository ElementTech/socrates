Socrates, Current Features
Define and Arrange your Code in every way you want.
Block
Block is a component made of a Code in a specific Language. It might have parameters, defaults, and a Pre-Code to execute before the main. Any files during execution will be saved as Artifacts, and Outputs can be set by printing following syntax: ::set-output key=value
Instance
Instance is attached to a Block. It is essentialy a Parameterized Block, with it's own execution History. Multiple Instance can exist for the same Block.
Steps Flow
Steps Flow is a pipeline of one or more Instances arranged in Steps. They share ENV Variables and Output to the next Steps. It has it's own execution history and can be drilled down to individual Instances.
DAG Flow
DAG Flow is a pipeline of one or more Instances arranged in Graph. They share ENV Variables, and pass Output to the next Nodes in the Branch. It has it's own execution history and can be drilled down to individual Instances.
Icons
Icons allow you to give Flows, Instances and Blocks an image on creation that will follow them wherever they are.
Shared Parameters
Parameters allow you to define shared Keys and Values which are fixed into Components. A change in a Shared parameter will propegate to all Flows, Instances and Blocks.
